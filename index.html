Estructura red en GNS3

GNS3 DHCP DNS Linux Windows

Diseño e implementación de una red empresarial simulada en GNS3 con maquinas virtuales de Windows y Linux. La red incluye segmentación por VLANs, servidor DHCP para la asignación dinámica de IPs, servidor DNS para la resolución de nombres internos y Zona Desmilitarizada. Con una gestion de sesiones de usuario a través de Windows Server con Active Directory

topología

Router Central con switches y 3 segmentos de red diferenciados.

Servicios

DHCP y DNS configurados en servidor Linux Debian

Seguridad

Reglas de Firewall entre segmentos, control de acceso y zona Demilitarizada

Lo que aprendi

  • Configuración de protocolos de enrutamiento
  • Gestión de servicios de red de Linux
  • Diagnóstico y resolución de problemas de red
